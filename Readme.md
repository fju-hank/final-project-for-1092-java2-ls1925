# 程式設計(二) 期末專題
## 曾欣茹 409570145 

## 專題用途
此專題有關一個投幣式洗衣機的介面收費程式，大型衣物50元/次 時長45分鐘，一般衣物30元/次 時長30分鐘，襪子20元/次 時長15分鐘。

## 使用方式
執行程式為 com.fju.Tester，按4:出現洗衣小撇步，7秒後連結到洗衣SOP網站，跳回來程式有選單，按a洗大型衣物，按b洗一般衣物，按c洗襪子，按0為進入結帳，結帳時請使用者輸入錢，會退還餘額，之後有警示訊息按1可看新竹市監視器資訊，最後按9有滿意度調查。


## 自行加入內容(補充說明)
一開始會問顧客有無想參考洗衣服的一些小撇步(可按4來觀看，按其他鍵略過)，會於7秒後跳出顯示洗衣的SOP網站，再來請顧客回程式區繼續執行之後動作，會列出所有選項問顧客要洗哪些，按a是洗大型衣物，按b是洗一般衣物，按c是洗襪子，可以重複選取(怕有洗一次洗不乾淨的問題)，等選取完選項，按0則可進行結帳之事(若於初始無選取任何選項時就按0，系統會認定為無想繼續選取之意，結束程式)，如果客戶按到其他非a,b,c,0之按鍵，會提醒客戶按錯之事，再來，會請顧客輸入投進去的錢(若餘額不足會通知客戶，並退還錢，結束程式)，成功結完帳後會顯示所選的選項、次數、耗時多長，為了維護洗衣店的安全，會有警示訊息讓顧客知曉，最後一步則是要請顧客按9連線另一視窗(JFrame)，詢問滿意度，按叉叉後，則結束全部程式。
